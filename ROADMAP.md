# Roadmap

Planned and proposed features for DayView. Each entry links to its design doc
where one exists. Nothing here is committed to a release until it ships.

## Proposed features

- **macOS Widget** — Show the day's remaining time as a native macOS widget
  (Notification Center / desktop), so the countdown is visible without opening
  the app. Design doc to follow.
- **Calendar-aware net time** — Subtract scheduled events from the raw day time
  to show a *net* available time. See
  [docs/calendrier-temps-net-plan.md](docs/calendrier-temps-net-plan.md).
- **Android ↔ Desktop sync** — Keep the same state across devices while staying
  usable offline. See [docs/device-sync-plan.md](docs/device-sync-plan.md).
- **Discord focus presence** — Optional, opt-out Discord integration for focus
  sessions. See
  [docs/discord-focus-presence-plan.md](docs/discord-focus-presence-plan.md).
- **Zoom camera mode** — See
  [docs/zoom-camera-mode-plan.md](docs/zoom-camera-mode-plan.md).
- **macOS virtual camera** — See
  [docs/macos-virtual-camera.md](docs/macos-virtual-camera.md).
